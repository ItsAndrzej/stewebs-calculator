# SteWeb's Ballistic Calculator

A ballistic calculator for airgun and pellet shooting. Built as a Progressive Web App (PWA) — install it on any phone or use it directly in the browser.

**Live app:** https://itsandrzej.github.io/stewebs-calculator/

---

## Features

- **Muzzle energy** calculation in Joules and ft·lbf
- **Dual unit support** — switch between m/s / fps and grains / grams
- **Preset pellets** — quick-fill common pellets with known caliber and weight
- **Favorites** — save and reload your own pellet configurations
- **Export / Import** favorites as a JSON file
- **Swedish legal indicator** — flags when energy exceeds the 10 J license threshold
- **SE / EN language toggle**
- **Offline support** — works without internet after first load
- **Installable** — add to home screen on Android and iOS

---

## Install on your phone

**Android (Chrome):**
1. Open the link above in Chrome
2. Tap the install banner or go to ⋮ → *Add to Home Screen*

**iPhone (Safari):**
1. Open the link above in Safari
2. Tap the Share button → *Add to Home Screen*
